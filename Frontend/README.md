# PanchKarma - Ayurvedic Customer Management (Frontend)

PanchKarma is a calm, modern frontend-only clinic management UI built with React, Vite, and Tailwind CSS. It supports Admin and Receptionist roles with role-based navigation, dashboards, and local storage persistence.

## Features

- Role-based login with demo credentials
- Admin dashboard with service stats and customer management
- Reception dashboard for adding and updating customers
- LocalStorage persistence (no backend)
- Responsive Ayurvedic-themed UI

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

- src/components: UI building blocks
- src/context: Auth and customer data
- src/pages: Login and dashboards
- src/utils: LocalStorage helpers
