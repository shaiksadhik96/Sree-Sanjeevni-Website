# Sree Sanjeevni Clinic Management System

A full-stack web application for managing daily clinic operations.

It supports two roles:
- Admin
- Receptionist

## Features

- Role-based login and protected routes
- Patient management
- Booking creation and admin approval workflow
- Discount request and approval workflow
- Billing and payment tracking
- Dashboard stats, reports, and notifications

## Tech Stack

Frontend:
- React (Vite)
- Tailwind CSS
- Framer Motion
- Recharts
- React Router

Backend:
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT authentication
